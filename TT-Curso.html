Java é uma linguagem de programação orientada a objetos e uma plataforma de desenvolvimento amplamente utilizada. Criada pela Sun Microsystems em 1995 e atualmente mantida pela Oracle, Java é conhecida por sua portabilidade, segurança e robustez.

Principais Características do Java

Portabilidade: "Escreva uma vez, execute em qualquer lugar" (WORA – Write Once, Run Anywhere), pois os programas Java rodam na JVM (Java Virtual Machine), tornando-os independentes do sistema operacional.

Orientação a Objetos: Segue o paradigma da Programação Orientada a Objetos (POO), permitindo reutilização de código e modularidade.

Segurança: Tem um forte mecanismo de gerenciamento de memória e proteção contra execução de código malicioso.
